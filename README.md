# kroton-tf

Pre-requisitos:

- Conta na AWS
- Console AWS configurado com acesso a conta (https://console.aws.amazon.com/)
- Terraform (https://www.terraform.io/downloads.html)

Usage:
Execute os passos a seguir:

- git clone https://github.com/Ellie-gg/kroton-tf.git
- cd kroton-tf
- terraform init 
- terraform plan
- terraform apply
